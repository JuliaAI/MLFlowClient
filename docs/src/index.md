## Installation

To install [MLFlowClient](https://github.com/JuliaAI/MLFlowClient.jl), start a `julia` session and run the following:

``` julia-repl
julia> import Pkg

julia> Pkg.add("MLFlowClient")
```

or simply:

``` julia-repl
]add MLFlowClient
```
